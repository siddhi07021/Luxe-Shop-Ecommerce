# Luxe Shop 👗

A modern e-commerce storefront designed for fashion enthusiasts.

## ✨ Features
- **Aesthetic UI**: Inspired by K-drama and C-drama fashion trends.
- **Responsive Design**: Optimized for mobile and desktop viewing.
- **SEO Ready**: Semantic HTML structure for better search engine visibility.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5.
- **Backend**: Python (Flask).
- **Design Style**: Minimalist & High-end.
